# 软件&网站导航

一个收录各种软件官方网站和优质网站的导航页面，方便快速查找和访问。部署于 GitHub Pages。

## 功能特点

### 🎯 核心功能
- **分类导航**: 支持多种软件和网站分类（浏览器、开发工具、设计工具等）
- **智能搜索**: 实时搜索软件名称、描述、标签和分类
- **响应式设计**: 完美适配桌面端和移动端设备
- **热门推荐**: 突出显示精选软件和网站
- **标签系统**: 通过标签快速筛选相关内容
- **类型区分**: 明确区分软件和网站两类资源

### 🛠️ 管理功能
- **可视化管理**: 直观的管理后台界面，支持添加、编辑、删除项目
- **数据导入导出**: 支持JSON格式数据的导入和导出
- **批量操作**: 支持批量添加、编辑和删除项目
- **灵活图标**: 支持Font Awesome图标类名和自定义图片URL
- **实时预览**: 添加/编辑项目时可实时预览效果
- **双模式管理**: 
  - 网页管理后台 (`admin.html`) - 可视化操作
  - 直接编辑 `data.json` - 适合技术人员

### 🔧 技术特性
- **纯前端实现**: 无需后端服务器，直接部署即可使用
- **现代化UI**: 采用简洁美观的设计风格
- **高性能**: 优化的渲染性能和用户体验
- **易扩展**: 模块化代码结构，便于功能扩展

## 数据格式

项目数据存储在 `data.json` 文件中，采用以下格式：

```json
[
  {
    "id": 1,
    "name": "项目名称",
    "description": "项目描述",
    "url": "https://example.com",
    "category": "分类名称",
    "tags": ["标签1", "标签2"],
    "icon": "fas fa-icon",
    "iconColor": "#颜色代码",
    "popular": false,
    "type": "software" // 或 "website"
  }
]
```

## 使用方法

### 🌐 访问页面
- **主页面**: [index.html](index.html) - 浏览和搜索软件网站
- **管理后台**: [admin.html](admin.html) - 管理项目数据

### 🛠️ 管理数据

#### 方式一：使用管理后台（推荐新手）
1. 打开 [admin.html](admin.html)
2. 点击"添加项目"按钮
3. 填写项目信息：
   - 项目名称和链接
   - 选择项目类型（软件/网站）
   - 选择分类和设置标签
   - 配置图标和颜色
4. 点击保存即可

#### 方式二：直接编辑JSON（适合技术人员）
1. Fork 此项目到您的 GitHub 账户
2. 编辑 `data.json` 文件
3. 提交更改并创建 Pull Request

### 🔍 搜索功能
- 在搜索框中输入关键词
- 支持搜索软件名称、描述、标签和分类
- 实时显示搜索结果

### 📱 响应式支持
- 完美适配手机、平板和桌面设备
- 触摸友好的交互设计
- 自适应布局和字体大小

## 管理后台详细说明

### 💻 界面概览
管理后台(`admin.html`)提供了完整的数据管理功能，包含以下主要区域：
- **项目列表区**: 显示所有已添加的软件/网站项目
- **表单编辑区**: 用于添加新项目或编辑现有项目
- **操作控制区**: 包含导入、导出、批量操作等功能按钮
- **实时预览区**: 实时展示当前编辑项目的显示效果

### ✨ 主要功能

#### 1. 添加项目
- 点击"添加项目"按钮进入新建模式
- 填写完整表单信息（必填项包括名称、URL、分类）
- 实时预览功能可即时查看图标显示效果
- 支持从Font Awesome选择图标或输入自定义图片URL

#### 2. 编辑项目
- 在项目列表中点击"编辑"按钮
- 修改任意字段内容
- 保存后自动更新数据并刷新列表

#### 3. 删除项目
- 点击项目旁边的"删除"按钮
- 系统会弹出确认对话框防止误操作
- 确认后立即从列表中移除

#### 4. 数据导入导出
- **导出数据**: 点击"导出JSON"按钮，将当前所有数据保存为JSON文件
- **导入数据**: 选择之前导出的JSON文件，一键恢复所有数据
- 数据文件格式与`data.json`完全兼容

#### 5. 批量操作
- 支持同时选中多个项目进行批量删除
- 可一次性导入多个项目数据
- 为团队协作和大规模数据迁移提供便利

#### 6. 实时预览
- 在编辑时右侧会实时显示项目卡片预览
- 包含图标、名称、描述等完整信息展示
- 确保数据填写的准确性和视觉效果

### 🎨 图标使用指南

#### Font Awesome 图标
1. 访问 [Font Awesome 官网](https://fontawesome.com)
2. 搜索需要的图标
3. 复制图标的CSS类名（如：`fas fa-chrome`）
4. 在图标字段中粘贴类名

#### 自定义图片图标
- 支持PNG、JPG、SVG等常见图片格式
- 推荐使用透明背景的正方形图片
- 最佳尺寸：64x64 到 128x128 像素
- 直接输入图片的完整URL地址

### 🚀 使用流程
1. 在浏览器中打开 `admin.html`
2. 开始添加第一个项目
3. 通过实时预览检查显示效果
4. 完成所有项目添加后，导出JSON数据
5. 将导出的数据替换项目根目录下的 `data.json` 文件
6. 部署更新后的文件到服务器

### ⚠️ 注意事项
- **数据安全**: 定期导出备份数据，避免意外丢失
- **ID管理**: 系统自动分配唯一ID，无需手动设置
- **字符编码**: 使用UTF-8编码确保中文正常显示
- **链接验证**: 确保所有URL链接有效可用
- **图片稳定性**: 自定义图片建议使用稳定的CDN链接
