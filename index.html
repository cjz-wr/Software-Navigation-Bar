JSON数据结构

在 data.json 文件中按照以下格式添加新的软件或网站：

[
  {
    "id": 1,
    "name": "软件/网站名称",
    "description": "详细描述",
    "url": "https://example.com",
    "category": "分类名称",
    "tags": ["标签1", "标签2"],
    "icon": "fas fa-icon",
    "iconColor": "#颜色代码",
    "popular": false,
    "type": "software" // 或 "website"
  }
]

常用的图标类：

  • fab fa-chrome (浏览器)
  • fas fa-code (开发工具)
  • fas fa-paint-brush (设计工具)
  • fas fa-video (视频会议)
  • fas fa-music (音乐)
  • fas fa-file (文档)
  • fas fa-blog (博客)
  • fas fa-book (文档/书籍)
